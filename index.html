How to make a page for the Web

Ingredients

  • HTML document
  • Colors
  • Typeface
  • Layout
  • Animation

Instructions

Add content

  1. Create a new HTML document..
  2. Add your text, images and URLs as plain text.
  3. Mark up your content to make it readable and interactive.
  4. Open your document in a Web browser, to see how it’s looking.

Style your content

  1. Create an empty file with a “.css” extension. This will be your style sheet.
  2. Add a link to the style sheet in your HTML document.
  3. Open the style sheet and change the colors, margins and typeface to your liking.
  4. (Optional) Add animation to make the page more lively.

Publish your page

Put your page on GitHub for the world to see!

Ingredients

HTML document

<!doctype html>
<html lang="en" dir="ltr">
<head>
  <meta charset="utf-8" />
  <title></title>
</head>
<body>

</body>
</html>

Copy and paste this into a new file. (Be sure to give it a “.html” extension.)

Headline

<h1>The name of this page or section</h1>

Paragraph

<p>A section of text that represents a connected thought</p>

Link

<a href="The URL">The name of the link</a>

Image

<img src="The URL" alt="A caption for the image" />

Style

<link href="style.css" rel="stylesheet" />

Place this in the “head” element of your HTML document.

Color

element {
  background: black;
  color: white;
}

Replace the word “element” with the name of the HTML element you want to target (for example “body” or “p”).

Typeface

element {
  font-family: Helvetica, sans-serif;
}

Layout

element {
  max-width: 40em;
  margin-left: auto;
  margin-right: auto;
}

Animation

By Justin Aguilar

@keyframes tossing {
  0% {
    transform: rotate(-4deg); 
  }
  50% {
    transform: rotate(4deg);
  }
  100% {
    transform: rotate(-4deg); 
  }           
}
element {
  animation-name: tossing;
  animation-duration: 2.5s; 
  animation-iteration-count: infinite;
}
